\## 📋 Project Overview



This system detects AI-generated deepfake images using transfer learning with ResNet50. It achieves 95.2% validation accuracy and provides a user-friendly web interface for real-time predictions.



\*\*Demo:\*\* Upload an image → Get instant "Real" or "Fake" prediction with confidence score



\## ✨ Features



\- 🖼️ Image upload (JPG, PNG, JPEG)

\- 🤖 AI-powered deepfake detection

\- 📊 Confidence score display (0-100%)

\- 🌐 Web-based interface (no installation needed)

\- ⚡ Fast predictions (<2 seconds)

\- 📝 Prediction history logging

\- 🔌 RESTful API for integration



\## 🛠️ Technology Stack



\### Backend

\- \*\*Python\*\* 3.10

\- \*\*TensorFlow\*\* 2.28 / Keras 3.12

\- \*\*Flask\*\* 2.3.0 (REST API)

\- \*\*SQLite\*\* (Database)

\- \*\*OpenCV\*\* 4.13 (Image processing)



\### Frontend

\- \*\*HTML5\*\* (Structure)

\- \*\*CSS3\*\* (Styling)

\- \*\*JavaScript ES6+\*\* (Interactivity)

\- No frameworks (vanilla JS)



\### Machine Learning

\- \*\*Architecture:\*\* ResNet50 (ImageNet pre-trained)

\- \*\*Training Platform:\*\* Google Colab (Free GPU)

\- \*\*Dataset:\*\* 10,000 images (FFHQ + StyleGAN)

\- \*\*Training Time:\*\* 30-40 minutes
